instafood has below features
1. create new menu "http://localhost:4200/pages/menu"
2. create order "http://localhost:4200/pages/order"
3. orders list "http://localhost:4200/pages/dashboard"

Create new menu -
It accepts input from users, Submit button has validation implemented, and will be disabled if no data is entered

Create order -
It is wizard based input form, accepts order from users, confirms order, accepts tips and performs virtual payment.

Order list - dashboard
All the placed orders will be available on the dashboard.
